Game of Battleships
School Project under ECS 36B

This program accepts 1 file argument that sets the parameters of the game:

Size of Y Axis
Size of X Axis
Number of Ships
Ship Length


For example:
10
10
5
C 5
B 4
D 3
S 3
P 2


"10,10" Creates a 10*10 Board, "5" is the number of ships, and "C, 5" is the ship "C" with the length 5.
